# JsonMin
**JsonMin is a minimized version of json.**

You can **write** and **read** json files.

supported variable format :
- int 
- float
- string
- bool
- char
- null

Before writing your code, you need to import the JsonMin library:

	#include "path/to/library/json.hpp"

Example **code** for **writing** :

	JsonNode node;

	int intV = 1234;
	bool boolV = true;
	char charV = 'h';
	float floatV = 1.08573f;

	node["str"] = "hello world!";
	node["int"] = toString(intV); // or "1234"
	node["bool"] = toString(boolV); // or "true"
	node["char"] = toString(charV); // or "h"
	node["float"] = toString(floatV); // or "1.08573"

	node.write("data.json", true); //replace 'data.json' with the name of your json file

**JsonNode::write(const char* path, bool isLisibleForHuman)**
with **isLisibleForHuman** to **true**:

	{
		"bool": true,
		"char": "h",
		"float": 1.08573,
		"int": 1234,
		"str": "hello world!"
	}

with **isLisibleForHuman** to **false**:

	{"bool": true, "char": "h", "float": 1.08573, "int": 1234, "str": "hello world!"}

Exemple **code** for **reading** :

	JsonNode node;

	node.read("data.json"); //replace 'data.json' with the name of your json file

	std::cout << node["int"] << std::endl;
	std::cout << node["str"] << std::endl;
	std::cout << node["float"] << std::endl;
	std::cout << node["bool"] << std::endl;
	std::cout << node["char"] << std::endl;

**output** :
	
	1234
	hello world!
	1.08573
	true
	h

you can later convert your variables to 
- int with the function toInt
- float with the function toFloat 
- bool with the function toBool
- char with the function toChar
